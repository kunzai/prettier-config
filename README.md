# `@kunzai/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:
```bash
$ npm install --save-dev @kunzai/prettier-config
```

```bash
$ yarn add --dev @kunzai/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@kunzai/prettier-config"
}
```
